# Titanic Survival Prediction

## Overview

This repository contains my machine learning experiments on Kaggle's Titanic: Machine Learning from Disaster dataset.

The goal is to predict passenger survival using different machine learning models and progressively improve performance through better algorithms and feature engineering.

## Models

### 1. Logistic Regression

* Baseline model
* Training Accuracy: 80.73%
* Test Accuracy: 79.21%
